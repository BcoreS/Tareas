Tarea #1 de Diseño Web

Motivo de la Tarea

Se ha solicitado que nosotros investigemos acerca de diferentes tipos de etiquetas HTML

¿Cuáles son esas etiquetas?

  • Etiquetas de estructura básica
  • Etiquetas de estructura de sección
  • Etiquetas de estructura de grupo
  • Etiquetas de texto
  • Etiquetas de enlace
  • Etiquetas de imagen
  • Etiquetas de miltimedia
¿Qué son?:
Son elementos pertenecientes a HTML que indican al navegador web el contenido que debe mostrar en la página

¿Para qué sirven?: 
Para indicar controles, contenido y demás en la página
web al navegador. 

¿Cómo funcionan?: En un documento de extensión html, se
colocan siguiendo una estructura para predefinir el contenido de una página
web, para el diseño se utiliza css

De esta forma, he experimentado con esta tarea, dandome cuenta de como se pueden trabajar diferentes etiquetas
